# Smart Support Copilot

AI-powered support assistant using:

- Retrieval-Augmented Generation (RAG)
- Query Classification
- Dynamic Routing
- Conversational Memory
- Streamlit UI

## Features

- PDF upload support
- Intelligent query classification
- Structured troubleshooting responses
- Product comparison support
- Conversational memory
- FAISS semantic retrieval

## Tech Stack

- Streamlit
- Azure OpenAI
- FAISS
- Sentence Transformers
- LangChain

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Sample Queries

- Why is my Galaxy phone overheating?
- Compare Galaxy S23 vs S24
- How do I reset Samsung Smart TV?
